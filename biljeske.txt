GRABEZLJIVAC-PLIJEN
- Lotka-Volterra: proucava odnosno dvije vrste
- 
- x = broj plijena, y = broj grabezljivca
- a = stopa rasta plijena kad nema grabezljivca, b = efikasnost stete grabezljivca plijenu
- c = stopa mortaliteta grabezljivaca, d = stopa rasta grabezljivaca u interakciji s plijenom
- dx/dt = x(a-b*y) 
- dy/dt = -y(c-d*x)

- RM dodaci:   a -> logisticki rast a(1-x/K)
               modifikacija predacije = d -> d/(x+e), b -> b/(x+e) 
- 

- sol = solve(problem) // rjesenje ODE
- plot!(sol, vars = (0, 2), label = "Grabezljivac") -> neka varijabla (y) ovisno o t
- plot!(sol, vars = (1, 2), label = "Grabezljivac") -> neka varijabla (y) ovisno o drugoj (x)
